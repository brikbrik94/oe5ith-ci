Sidebar Reference

Vollständige Referenz aller Sidebar-Elemente. Die Sidebar links ist die Live-Demo — alle Elemente sind funktional. Breakpoints, Footer-Typen und Accordion-Status live testbar.

Verfügbare Elemente
Nav-Item
Interner Link mit Icon. Aktiv: Accent-Farbe + border-left. Optionaler Status-Dot rechts.
Externer Link
Wie Nav-Item, aber color #666 und Pfeil-Icon rechts. Kein aktiver Zustand. Öffnet neuen Tab.
Accordion-Gruppe
Aufklappbar. Dot (site-spezifische Farbe), Titel, Status-Badge, Checkboxen, Schnellzugriff.
Filter-Feld
Texteingabe filtert Accordion-Gruppen nach Name. Optional — nur bei >5 Gruppen sinnvoll.
Section-Label
0.65rem · uppercase · color #555. Nur bei mehreren Gruppen anzeigen — nie für eine einzelne Gruppe.
Trenner
1px Linie aus --border. Zwischen logisch getrennten Bereichen. Kein Label nötig.
Status-Dot (Nav)
7px Dot rechts im Nav-Item. online (--success) / offline (#ef4444) / unbekannt (#444).
Footer
Immer unten sichtbar. Links: Version (Mono). Rechts: Status-Dot + Text. Drei Seitentypen.
