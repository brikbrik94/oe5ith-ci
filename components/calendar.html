Mobile — Show-All-Toggle
Auf Mobile (≤768px) werden leere Tage standardmäßig ausgeblendet. Mit .calendar--show-all auf .calendar werden alle Tage angezeigt. Die Klasse wird von der Website per JS gesetzt — der Button im Monatskalender-Header demonstriert dieses Verhalten (nur auf Mobile sichtbar wirksam).
Standard (kompakt):
<div class="calendar">
Alle Tage sichtbar:
<div class="calendar calendar--show-all">
