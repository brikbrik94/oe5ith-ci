# Calendar Component Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Kalender-Komponente (Monatsansicht) für den Dienstplan im Internal Dashboard — read-only, farbcodiert nach Diensttyp, Detailansicht per Modal.

**Architecture:** Freistehender Content-Block ohne Card-Wrapper. Eine neue `css/calendar.css` mit allen Regeln, eine `components/calendar.html` Referenzseite und `docs/calendar.md`. Das bestehende `.modal-backdrop`/`.modal` aus `modal.css` wird für die Detailansicht wiederverwendet. Keine neuen Tokens — alle Werte kommen aus `common.css`.

**Tech Stack:** Vanilla CSS (Custom Properties), Vanilla HTML5, Font Awesome 6, kein Build-Schritt.

---

| Datei | Aktion | Verantwortung |

function openCalModal(entry) {
  const backdrop = document.getElementById('cal-modal-backdrop');
  document.getElementById('cal-modal-title').textContent = entry.dataset.title || '';
  document.getElementById('cal-modal-date').textContent  = entry.dataset.date  || '';
  document.getElementById('cal-modal-time').textContent  = entry.dataset.time  || '';
  document.getElementById('cal-modal-desc').textContent  = entry.dataset.desc  || '';

  const badge = document.getElementById('cal-modal-badge');
  badge.className = 'badge ' + (entry.dataset.badgeClass || 'badge-blue');
  badge.textContent = entry.dataset.badgeLabel || '';

  const changedBlock = document.getElementById('cal-modal-changed');
  if (entry.dataset.changed) {
    document.getElementById('cal-modal-changed-date').textContent = entry.dataset.changed;
    document.getElementById('cal-modal-changed-note').textContent = entry.dataset.changedNote || '';
    changedBlock.style.display = 'flex';
  } else {
    changedBlock.style.display = 'none';
  }

  backdrop.querySelector('.modal-close').focus();
}

function closeCalModal() {
  document.getElementById('cal-modal-backdrop').classList.remove('open');
}

function handleModalBackdrop(e) {

| Modifier-Klasse | Diensttyp | Tokens |
